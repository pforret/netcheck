# testnetwork

test network config for problems

## Installation

with [basher](https://github.com/basherpm/basher)

	$ basher install pforret/testnetwork

or with `git`

	$ git clone https://github.com/pforret/testnetwork.git
	$ cd testnetwork

## Usage

	> testnetwork.sh [action]
